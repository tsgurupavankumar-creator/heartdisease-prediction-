# Heart Disease Prediction Platform

A Streamlit-based healthcare analytics application for predicting heart disease risk using machine learning. The platform supports dataset exploration, data quality checks, exploratory analysis, model training, evaluation, comparison, feature importance, and real-time patient risk prediction.

## Features

- Interactive Streamlit dashboard
- Heart disease risk prediction from clinical indicators
- Manual patient data entry
- PDF, DOCX, TXT, and CSV document upload autofill
- Dataset exploration and data quality reporting
- Model training, evaluation, and comparison
- Feature importance and patient-specific risk factor insights
- Risk probability gauge and confidence scoring

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Plotly
- pypdf
- python-docx

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

Then open:

```text
http://localhost:8501
```
